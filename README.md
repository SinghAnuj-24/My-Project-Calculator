# My-Project-Calculator
My first project using python.
This is a simple Digital Calculator built using Python. It performs basic arithmetic operations such as addition, subtraction, multiplication, and division. This project was created as a beginner-level Python project to practice programming fundamentals.

## Features
- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- User-friendly command-line interface
- Handles basic arithmetic calculations
